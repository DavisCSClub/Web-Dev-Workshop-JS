Two ways to custom style elements

By ID's

         <p id="sample_paragraph">This is a sample paragraph</p>
           

         /* Then in the CSS */
         #sample_paragraph {
           color: red;
         }

By Classes


        <ul class="squareList">
          <li>Bla1</li> 
          <li>Bla2</li>
        </ul>
                

        /* Then in the CSS */
        .squareList {
          list-style-type: square;
        }
